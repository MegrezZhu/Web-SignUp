##Instruction
1. Install dependencies before launching.
    
    ```npm
    npm install
    ```
2. To launch the Node.js server:

    ```npm
    npm start
    ```
    or
    
    ```npm
    node server
    ```
3. This server listens at port `8000`. Use `localhost:8000` for accessing.
